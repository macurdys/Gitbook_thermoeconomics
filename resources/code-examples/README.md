# Code Examples

Runnable code for exploring thermoeconomic concepts. All examples are provided in both **Wolfram Language** and **Python**.

---

---

## Running the Code

**Wolfram Language** — Runs in [Wolfram Cloud](https://www.wolframcloud.com/) (free account) or Mathematica desktop. Files have `.wl` extension.

**Python** — Runs anywhere with Python 3.8+. Uses standard libraries: `hashlib`, `numpy`, `matplotlib`. Files have `.py` extension.
